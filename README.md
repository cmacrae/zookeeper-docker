# Docker ZooKeeper
A super simple Docker container to run Apache ZooKeeper

## Exposed variables
| Key | Default Value   |
|-----------------------|
| ZOOKEEPER_TICKTIME   | 2000 |
| ZOOKEEPER_INITLIMIT  | 10   |
| ZOOKEEPER_SYNCLIMIT  | 5    |
| ZOOKEEPER_CLIENTPORT | 2181 |

## Current ZooKeeper Version
`3.4.9`
